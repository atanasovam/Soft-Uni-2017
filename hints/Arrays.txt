for (int i = 0; i < count % array.Count; i++)

Завъртане на масива надясно "n" пъти.

for (int i = 0; i < count % array.Count; i++)
            {
                var temp = array[array.Count - 1];
                array.RemoveAt(array.Count - 1);
                array.Insert(0, temp);
            }
